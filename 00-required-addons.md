# Required add-ons:

[VRM Format](https://extensions.blender.org/add-ons/vrm/) (Base tools for VRM stuff)

[Blender MMD Tools](https://extensions.blender.org/add-ons/mmd-tools/) (Base tools for working with MMD file formats)

[MMD Tools Append](https://github.com/MMD-Blender/blender_mmd_tools_append) (Additional MMD tools)
