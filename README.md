# VRoid -> MMD in Blender resources

This is a repository of my observations, techinques, and thoughts on working with VRoid (`.vrm`) models with MMD animations and such.

## Usage

Use this repository simply as an online resource, reading through the markdown files as you need. You can also `git clone` this repository if you want an offline experience. If you only want specific scripts, go to any script in this repository's hierarchy, then click the download icon that says "Download raw file."
